# 🤖 AI Tic-Tac-Toe

An interactive Tic-Tac-Toe game where a human player competes against an unbeatable AI opponent powered by the **Minimax Algorithm**. The project features a modern, colorful, and responsive user interface built using HTML, CSS, and JavaScript.

---

## 📌 Project Overview

This project demonstrates the implementation of Artificial Intelligence in a classic game environment. The AI uses the **Minimax Algorithm** to analyze all possible future moves and select the optimal move, making it impossible to defeat.

The project was developed as a beginner-friendly introduction to:

- Game Theory

## 🧠 AI Algorithm

### Minimax Algorithm

The AI uses the Minimax algorithm, a decision-making algorithm commonly used in turn-based games.

The algorithm:

1. Simulates all possible future moves.
2. Evaluates each game state.
3. Chooses the move that maximizes the AI's chance of winning.
4. Assumes the human player always makes the best possible move.

Because every possible game state is analyzed, the AI cannot be beaten.

### Scoring System

| Game Result | Score |
|------------|--------|
| AI Wins | +10 |
| Human Wins | -10 |
| Draw | 0 |

## 🎯 How to Run the Project

### Prerequisites

- Visual Studio Code
- Live Server Extension

### Steps

1. Download or clone the repository.

```bash
git clone <repository-url>
```

2. Open the project folder in VS Code.

3. Install the Live Server extension.

4. Open `index.html`.

5. Right-click and select:

```text
Open with Live Server
```

6. The application will launch in your browser.

```text
http://127.0.0.1:5500
```

---

## 🎮 How to Play

1. You play as **X**.
2. The AI plays as **O**.
3. Click any empty cell to make your move.
4. The AI immediately calculates and performs its move.
5. Continue until:
   - You win
   - AI wins
   - The game ends in a draw

6. Click **New Game** to restart.
